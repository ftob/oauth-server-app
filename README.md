### Getting start
===============================
1. Create environment.yml
```
environment:
    db:
      POSTGRES_DB: "oauth"
      POSTGRES_USER: "oauth"
      POSTGRES_PASSWORD: "oauth"
    deploy:
      DEPLOY_PRODUCTION_SERVER: "localhost"
      DEPLOY_PRODUCTION_USERNAME: "root"
      DEPLOY_STAGING_SERVER: "localhost"
      DEPLOY_STAGING_USERNAME: "root"
```
